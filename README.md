# BookExplorer

## Project Structure
- /scraper → Scraping books from books.toscrape.com
- /backend → REST API using Node.js & Express
- /frontend → React + TypeScript frontend

## Setup

### 1. Install dependencies
```bash
# Scraper
cd scraper
npm install
npm start (optional)

# Backend
cd ../backend
npm install
npm start

# Frontend
cd ../frontend
npm install
npm run dev
